# Network Access Protection (NAP)
# Microsoft technology for controlling network access of a computer, based on its health.
# With NAP, system administrators of an organization can define policies for system health requirements.
# Examples: has updates installed, has the latest anti-virus, has firewall enabled.
# https://en.wikipedia.org/wiki/Network_Access_Protection
